# euler-ts

> My solution to project Euler in Typescript

[![NPM](https://img.shields.io/npm/v/euler-ts.svg)](https://www.npmjs.com/package/euler-ts) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save euler-ts
```

## Usage

```tsx
import * as React from 'react'

import MyComponent from 'euler-ts'

class Example extends React.Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
